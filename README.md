## 简介
这是一个题材为 Android Studio 通过发射JetBrains系列的IDE来攻击各种版本的Eclipse的打飞机游戏。 :joy:<br/>
欢迎bug反馈。

## 安装
1. 你需要一部Android手机。
1. 下载[安装包](./app/app-release.apk)或克隆源代码并使用SDK编译。
1. 拷贝安装包到手机上 安装

## 操作方式
1. Android Studio会和你的手指同步位移。
